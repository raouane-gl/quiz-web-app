# Online Quiz Web Application

This project is a web-based quiz application that allows users to answer questions and receive scores.

## Technologies Used

* HTML
* CSS
* Tailwind CSS
* PHP
* SQL

## Features

* User can take a quiz and submit answers
* Automatic score calculation
* Database integration to store questions and answers
* Simple and responsive user interface

## Project Structure

* Frontend: HTML, CSS, Tailwind
* Backend: PHP
* Database: MySQL

## What I Learned

* Connecting frontend with backend using PHP
* Working with databases (SQL)
* Handling user input and data processing
* Building a complete web application

## Future Improvements

* Add user authentication (login/register)
* Improve UI/UX design
* Add timer for quiz
* Store user scores history
